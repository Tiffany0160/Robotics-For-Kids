# Robotics for Kids - Android App and Website

## Project Overview
"Robotics for Kids" is an interactive educational platform designed to teach children the basics of robotics. The project includes both a mobile application (Android) and a web-based interface that feature lessons, activities, and quizzes aimed at engaging kids in learning robotics. The application and website guide kids through various lessons, allow them to track progress, and provide feedback through quizzes and certificates of completion.

---

## Features

- **Home Screen:** A colorful and interactive layout tailored for children, with easy navigation.
- **Learning Modules:** List of robotics lessons with icons for easy identification.
- **Activity Tracker:** Track progress and completed modules.
- **Quizzes:** Fun tests and challenges to help children review what they've learned.
- **Certificates:** Award certificates upon quiz completion.
- **Contact Form:** For feedback and inquiries.

---

## Tech Stack

- **Frontend:**
  - Android App (Java)
  - HTML, CSS, JavaScript for Website
- **Backend:**
  - Firebase for user authentication and data storage
  - Azure DevOps for project management, CI/CD
- **Database:**
  - Azure SQL Database for storing user data, progress, and quizzes
- **Version Control:**
  - GitHub for source code management
- **CI/CD:**
  - Azure Pipelines for automated build and deployment
